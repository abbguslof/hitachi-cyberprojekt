# ABB - TheCrypticWebsite

En CTF uppgift för skolan. Deltagarna ska hitta och knäcka ett python-script för att få flaggan.

Länk till hemsidan: [TheCrypticWebsite](https://gustavs-ctf.webflow.io)

### Dependencies

- Python
- Webläsare
